# Lisensi Open Source

FluxLab dibangun dengan dependensi langsung proyek berikut. Setiap paket tetap tunduk pada lisensi dan pemberitahuannya sendiri. Dependensi transitif dapat menyediakan pemberitahuan tambahan melalui distribusi upstream masing-masing.

## AndroidX dan Jetpack Compose

AndroidX Core KTX, Activity Compose, Lifecycle, Navigation Compose, DataStore, Metrics, Room, Compose UI, Compose Material 3, dan Compose Material Icons didistribusikan berdasarkan Apache License, Versi 2.0.

## Kotlin dan Kotlin Coroutines

Kotlin dan kotlinx.coroutines didistribusikan berdasarkan Apache License, Versi 2.0. Pemberitahuan compiler dan runtime Kotlin tetap mengikuti distribusinya masing-masing.

## JUnit

JUnit digunakan untuk pengujian JVM deterministik dan didistribusikan berdasarkan Eclipse Public License, Versi 1.0.

## Teks lisensi
